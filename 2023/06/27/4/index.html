表格

使用

基本标签
标签名 说明
table 表格整体，可用于包裹多个tr
tr 表格每行，可用于包裹td
td 表格单元格，可用于包裹内容

代码：

<body>
  <table>
      <tr>
         <td>单元格内容</td>
         <td>单元格内容</td>     
      </tr>
      <tr>
          <td>第二行单元格内容</td>
          <td>第二行单元格内容</td>
      </tr>
  </table>
</body>

注意点：

  • 标签的嵌套关系：table>tr>td

表格的相关属性

场景：设置表格基本展示效果

常见相关属性
属性名 属性值 效果
border 数字 边框宽度
width 数字 表格宽度
height 数字 表格高度

注意点：

  • 实际开发时针对于样式效果推荐使用CSS

代码：<table border=" " width=" " heigth=" "></table>

表格的标题和表头单元格标签

场景：在表格中表示整体大标题和一列小标题

其他标签
标签名 名称 说明
caption 表格大标题 表示表格整体大标题，默认在表格整体顶部居中显示
th 表头单元格 表示一列小标题，通常用于表格第一行，默认内部文字加粗居中显示

注意点：

  • caption标签写在table标签内部
  • th标签书写在tr标签内部(用于替换td标签)

表格的结构标签（了解）

场景：让表格的内容结构分组，突出表格的不同部分（头部，主体，底部），使语义更清晰。

结构标签
标签名 名称
thead 表格头部
tbody 表示主体
tfoot 表示底部

注意点：

  • 表格结构标签内部用于包裹tr标签
  • 表格的结构标签可以省略

合并单元格

场景：将水平或垂直的多个单元格合并。

合并单元格步骤：

  1. 明确合并哪几个单元格
  2. 通过左上原则确定保留谁删除谁
    • 上下合并只保留最上的，删除其他
    • 左右合并只保留最左的，删除其他
  3. 给保留的单元格设置：跨行合并和跨列合并
属性名 属性值 说明
rowspan 合并单元格的个数 跨行合并，将多行的单元格垂直合并
colspan 合并单元格的个数 跨列合并，将多列的单元格水平合并

表单

input标签

代码:<input type=" ">

type 说明
text 文本框，用于输入单行文本
password 密码框，用于输入密码
radio 单选框,多选一
checkbox 多选框，多选多
file 文件选择，用于上传文件
submit 提交按钮，用于提交
reset 重置按钮，用于重置
button 普通按钮，无特殊功能,之后配合js添加功能

占位符：placeholder,用于提示用户输入。
代码:<input type=" " placeholder=" ">

input单选框

场景：在网页中显示多选一的单选表单控件

type属性值：radio。

属性名 说明
name 分组，有相同name属性值的单选框为一组，一组中同时只能有一个被选中
checked 默认选中

代码：<input type="radio" name=" " checked（添加后会默认勾选该选项）>

input其他属性值说明

多文件选择

代码：<input type="file" multiple>

按钮

submit:<input submit value=" ">
reset: <input reset value=" ">
button: 

<button type="submit">提交按钮</button>
<button type="reset">重置按钮</button>
<button type="button">普通按钮</button>
