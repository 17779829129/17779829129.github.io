选择器

标签

以标签名命名的选择器,选中所有的这个标签都生效CSS。

类

结构：.类名{ CSS属性名：属性值}
作用：通过类名，找到页面中所有带这个类名的标签
注意点：

  1. 所有标签上都有class属性，class属性的属性值为类名(类似于名字)
  2. 类名可以有数字，字母，下划线组成，但不能以数字和中划线开头
  3. 一个标签有多个类名，类名之间用空格分开
  4. 类名可以重复，一个类选择器可以同时选中多个标签

id

结构：#属性名{ CSS属性名：属性值}
一个网页中只能存在一个id

通配符

结构:*{CSS属性性名：属性值}
通配符会作用在该网页的全部标签上。

文字样式

字体大小：font-size
取值：大小+px；
字体粗细：font-weight
取值：

  • 关键字：
    • 正常 normal
    加粗 blod
  • 纯数字：100-900的整百数：
    • 正常 400
    加粗 700

font复合属性

属性名：font(复合属性)
取值：font:style weight size family
只能省略前两个，相当设置了默认值

文本标签

文本缩进

属性名：text-indent
取值：

  • 数字+px
  • 数字+em(推荐：1em=当前标签的font-size的大小)

代码：

p{
    text-indent:
}

水平对齐

属性名：text-align
取值：

属性值 效果
left 左对齐
center 居中对齐
right 右对齐

文本修饰

属性名：text-decoration
取值：

属性值 效果
underline 下划线(常用)
line-through 删除线
overline 上划线
none 无装饰线

拓展

水平居中：margin:0 auto

层叠性：同一属性值，后写的会覆盖先写的。
